# Subscription Template

A polished Marzban dashboard template collection maintained by **YrustPd**. The first entry, **AuroraGlass**, delivers a modern glassmorphism experience, dynamic subscription telemetry, and one-tap client onboarding across platforms.

## Templates

- [AuroraGlass](marzban/AuroraGlass/README.md) – Flagship dashboard skin with animated usage meters, adaptive status styling, QR importing, and deep links for iOS/Android/desktop clients.

## Highlights

- **Status aware UI** – Active, Limited, Expired, and Blocked accounts are colour-coded with graceful fallbacks.
- **Data intelligence** – Unlimited and capped plans receive tailored meters, with automatic byte formatting.
- **Time awareness** – Subscriptions display a live countdown with smart defaults and an "Endless" badge when no expiry is present.
- **One-click onboarding** – Fill-in-place URL templates launch Sing Box, V2Box, v2rayNG, and Hiddify using absolute links derived from your deployment.
- **Accessibility touches** – Keyboard-friendly navigation, focus-managed modals, and `prefers-reduced-motion` safeguards.

## Using the Templates

Each template lives in its own folder under `marzban/`. Follow the installation steps inside the respective README to deploy.
